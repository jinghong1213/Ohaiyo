# assets/

Drop `cat.gif` here to replace the ASCII cat in the splash window with an
animated GIF. Suggested sources:

- https://giphy.com/search/pixel-cat
- https://tenor.com/search/cat-pixel-gifs

Smaller files load faster — aim for under 200 KB. The splash window is
360x220 px; GIFs around 96x96 read well at that size.
